Star Wars Blog

A Front-End blog that illustrates Star Wars details. The application utilizes an API to display information on characters, planets and star-ships general information on the films from the official website for any fan to enjoy. It uses props and context and the top level in order to dynamically fetch and pass data to its children.

The main objective for this project was to apply a communication with an API. I also intergrated a favorites button on bootstrap cards in order to add them onto a "cart" placed on the navbar to replicate the use of a shopping cart - commonly used in typical e-commerce applications.
